# **App Name**: Nutrify

## Core Features:

- User Authentication & Profile: Secure user registration, login, and password management (forgot password, Google Sign-In) to establish a personalized account for tracking. This also includes basic profile information for initial setup.
- Personalized Onboarding Wizard: A multi-step guided setup process for new users to input personal details (gender, age, height, weight), set nutrition goals (lose/maintain/gain weight, eat healthier), and define dietary preferences and activity levels. This data is crucial for personalized tracking and recommendations.
- AI Food Recognition & Analysis: Enable users to upload images of their food, leveraging an AI tool to identify items, provide confidence levels for top predictions, and display nutritional data. Confirmed selections lead to a detailed food entry.
- Food Search & Nutrient Details: A comprehensive search functionality with filters (e.g., local Ghanaian, fruits, proteins) to browse a food database. Users can view detailed nutritional breakdowns (calories, macros, key micronutrients) and select portion sizes before adding to their tracker.
- Daily Nutrition Tracker: An organized interface allowing users to log food items into structured meals (Breakfast, Lunch, Dinner, Snacks). It displays cumulative calories and macronutrient distribution for the day, with options to remove entries.
- Personalized Dashboard Overview: The central hub displaying key daily nutritional progress. Includes summary cards for calories consumed/remaining, macro progress, and status indicators for key micronutrients like Iron and Vitamin A. It also features a weekly calorie intake chart and 'Smart Insights' based on user data.
- Goal Management: Allow users to review and adjust their daily calorie targets and customize macronutrient percentages (protein, carbs, fats) and specific micronutrient targets (e.g., Iron, Vitamin A) to align with evolving health objectives.

## Style Guidelines:

- Color Anchor: Health, natural Ghanaian vibrance, and 'smart' technology. The app uses a light color scheme for clarity and a clean feel.
- Primary Color: A sophisticated forest green (#29A368) for a fresh, natural, and trustworthy feel. (HSL: 150, 60%, 40%)
- Background Color: A soft, heavily desaturated light green (#F2F8F5) to maintain consistency and provide a clean canvas. (HSL: 150, 15%, 95%)
- Accent Color: A vibrant, slightly yellow-green (#7AE029) used for call-to-action elements, interactive states, and highlights, providing energetic contrast. (HSL: 120, 70%, 55%)
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and objective aesthetic, ensuring high readability across data displays and editorial content. This choice reinforces the 'smart' aspect of the platform.
- Utilize clean, modern outline icons across the application for elements like 'How It Works' steps, food categories, and macro indicators, ensuring a cohesive and contemporary visual language.
- Implement a modular and responsive grid-based layout for consistent component arrangement, optimizing user experience across various screen sizes. Mobile layouts will feature a drawer navigation for the sidebar and vertically stacked content for charts and planners.
- Integrate Framer Motion for smooth and engaging animations, specifically for page transitions, interactive card hover effects, fluid onboarding step progression, and confirmation states, to enhance user interaction and feedback.
